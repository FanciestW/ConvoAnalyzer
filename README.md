## Requirements:
- Python 3.6 or 3.7
- Pip3

## Usage:
1. Navigate into the project root directory.
2. Install all pip package dependencies by running:\
   ```pip install -r requirements.txt```
3. Run the program:\
   ```python3 main.py```
